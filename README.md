- 완성된 파일은 1차_가공 폴더나 2차_가공 폴더에 넣기

- 1차 가공 파일, 2차 가공 파일

    - 건형: 5
- 대학교마다 처리해야하는 함수 v1, v2
    - 지현, 민수
